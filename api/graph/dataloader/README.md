# dataloader

# コード生成の手順
dataloader配下でdataloadenを実行するとコード生成される

```
dataloaden TagLoader int '*my-go-gql-sample/api/graph//model.Tag'
```

# Warning
- 生成されたファイルのモデルは、importパスがついていないので、編集する必要がある
